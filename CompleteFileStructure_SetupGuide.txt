## ğŸ“ FILES YOU NEED TO UPDATE

### When Adding Instructor Resources:

1. **Update `src/pages/drones/instructor.astro`:**
   - Add new resources to `instructorResources` array
   - Add new items to `additionalResources` array

2. **Add files to `public/downloads/`:**
   - Place all ZIP files in this directory
   - Use exact filenames as specified in instructor page

3. **No changes needed to:**
   - Sidebar.astro (automatically includes all chapters)
   - Header.astro (navigation is static)
   - Quiz.astro (works with any quiz.json)

### When Adding New Lessons:

1. **Create content file:** `src/content/drones/XX-chapter/lesson-N.md`
2. **Create page file:** `src/pages/drones/XX-chapter/lesson-N.astro`
3. **Update Sidebar.astro:** Add new lesson to chapter's lesson array
4. **Update LessonNav:** Set correct prev/next links in new lesson page
5. **If adding quiz:** Create `quiz.json` and import in instructor.astro

## ğŸ” Troubleshooting

**Build errors?**
- Check all frontmatter is valid YAML
- Ensure all imported files exist
- Run `npm run build` to see specific errors

**Images not loading?**
- Ensure images are in `public/images/`
- Use `/images/filename.jpg` in markdown

**Quiz not working?**
- Check JSON syntax in quiz.json files
- Ensure correct index numbers (0-based)

**Instructor page not showing resources?**
- Verify file paths in downloads array
- Check that files exist in `public/downloads/`
